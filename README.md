# IoT Weather Reporting System

## 📌 Project Description
This project monitors temperature and humidity using ESP32 and DHT11 sensor. Data is sent using MQTT and displayed on Node-RED dashboard.

## 🛠 Technologies Used
- ESP32
- DHT11 Sensor
- MQTT Protocol
- Node-RED

## 🔗 Features
- Real-time monitoring
- Dashboard visualization
- Alert system (optional)

## ▶️ How to Run
1. Upload ESP32 code
2. Connect DHT11 sensor
3. Run Node-RED
4. Import flow.json
5. Open dashboard
